SharedPreferences

讲 Xposed 中的 XSharedPreferences 之前先讲一下 Android 中的 SharedPreferences，SharedPreferences 是Android 平台上一个轻量级的存储类，用来保存应用的一些常用配置 ，以 .xml 文件的形式保存在 /data/data/PACKAGE_NAME/shared_prefs/ 目录下，以类似键值对的方式来存储数据。

XSharedPreferences

Xposed API 中提供了一个 Xposed 独有的储存类叫 XSharedPreferences，这个类可以让你的 Xposed 模块在运行时读取一些配置数据以达到实现不同的功能。

Matters

出于安全性的考虑，Android SharedPreferences 中的 MODE_WORLD_READABLE 和 MODE_WORLD_WRITEABLE 在 Android 4.2 版本中已经被弃用，现在默认都使用 Context.MODE_PRIVATE，不过这样就 XSharedPreferences 就会没有权限读取 xml 文件，使用上面 Demo 中的 setWorldReadable() 方法可解决这一问题，原理就是 file.setReadable(true, false) 和 file.setExecutable(true, false) 来使文件对外界具有可访问性，当初在这个地方绕了好多弯路特此记下来。
