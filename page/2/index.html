Java 中的运算符

发表于 2017/11/24 更新于 2023/03/09 分类于 Java 基础 阅读次数：

Java 中的运算符可以这样分为六类：

  1. 算数运算符：+、-、*、/、%、++、--
  2. 关系运算符：==、!=、<、<=、>、>=
  3. 逻辑运算符：&&、||、!、^、&、|
  4. 位运算符：&、|、~、^、<<、>>、>>>
  5. 赋值运算符：=、+=、-=、*=、/=、%=、&=、|=、^=、<<=、>>=、>>>=
  6. 其他运算符：.、[]、()、?:、instanceof
